## 📖 用户操作指南

### 1. 探索工具
在首页您可以浏览所有已通过审核的优质脚本。通过顶部的分类标签（AI工具、Python相关等），快速锁定您需要的资源。

### 2. 获取脚本
点击对应的脚本卡片，即可一键下载原始文件或获取在线部署指令。

### 3. 分发您的脚本
1. **注册/登录**：点击右上角加入社区。
2. **上传脚本**：填入脚本标题、描述并上传物理文件。
3. **等待审核**：管理员审核通过后，您的脚本将立即在首页展示，造福全网极客。
